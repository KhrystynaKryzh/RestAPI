Shopping App Rest Api
RESTFul api service / WebAPI for online shopping for owners and customers, 
having CRUD operations, 
API allows CUSTOMERS to :
view products list, 
order online, 
track orders, 
view history, 
register as a member, 
view invoice 
and complete history since they registered, 
ADMIN can:
add new products, 
dilivery status, 
payment status
